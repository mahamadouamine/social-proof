#social-proof

##Page d’avis clients

Ce projet est une page web qui affiche des témoignages de clients satisfaits.  
Elle est faite en HTML et CSS uniquement.  
La page contient un grand titre, une description, et plusieurs cartes d’avis.  
Chaque carte montre le nom du client, sa photo et son commentaire.  
En haut de la page, des étoiles affichent la note donnée aux services.  
Le design est responsive: il s’adapte à la taille de l’écran.  
Sur mobile, les éléments se placent les uns sous les autres et sont centrés.  
Le CSS utilise Flexbox pour organiser et aligner les éléments facilement.  
Chaque bloc change légèrement de couleur au survol (effet hover).  
Ce projet m’a permis de pratiquer la mise en page, la responsivité et les media queries.  
C’est un bon exercice pour apprendre les bases du HTML/CSS.
